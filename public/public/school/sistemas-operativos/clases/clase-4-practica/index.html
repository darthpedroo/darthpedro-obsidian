Pasaje por copia

En C los parametros de las funciones son pasados por copia. Esto significa que la función..???

Scope !

Hay que usar referencias para resolver el problema de swapear dos variables en C++.

Para pasar por referncia se usa &

void swap(int &a, int &b){
	int temp = a;
	a = b;
	b = temp;
}

Se prefiere esto en vez de devolver un valor.

Streams

En c++ un stream es un objeto que representa un flujo de datos. -> Una fuente de entrada como un destino de salida de datos de nuestro programa.

Input Stream: Leer datos desde una fuente Output Stream: Escribir datos hacia un destino

cin -> Leer desde la entrada estandar cout -> Escribir en la salida estandar cerr -> Escribir en la salida de error estandar.

<< // escribir datos
>> // se utiliza para leer datos

»

  • “»” -> Lee hasta el primer espacio en blanco (Espacio, tab, enter, etc)

  • Intenta leer un tipo de archivo especifico

  • La expresion se evalua como true o false. Si llegó al final vale false, sino vale true.

    • EJ: Leer todas las palabras del archivo.

  • eof -> Verifica si llegamos al final del stream.

  • Si lo leo dos veces, tampoco guarda el delimitador. Entonces retoma desde donde había dejado.
