CPU utilization: Maximizar el uso del CPU

Throughput: Trabajos terminados por unidad de tiempo.

Turnaround Time: Tiempo total de ejecución de una tarea, desde que es lanzada en el sistema.

Waiting Time: Tiempo TOTAL de una tarea en la coal de listos

Response Time: Tiempo transcurrido desde que llegó el proceso a estar listo hasta que se ejecutó. (Cuanto tardó el chrome en abrir)

Algoritmos de Scheduling

First Come , First Served

  • Sencillo de implementar
  • La tarea que pide el CPU primero se la queda. El resto es por orden de llegada
  • No es Preemptive
    • Si se me cuelga la compu cagaste

Shortest Job First

Cuando el CPU se libera elige aquella tarea con ráfaga de CPU más chica.
